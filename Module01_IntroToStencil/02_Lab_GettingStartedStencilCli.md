# Getting Started with the Stencil CLI

## Install Stencil CLI

**Install:**

```shell
npm install -g @bigcommerce/stencil-cli
```

**Test stencil:**

```shell
stencil -h
```

## Clone the Cornerstone Theme

**Clone:**

```shell
git clone https://github.com/bigcommerce/cornerstone.git
```

**Install dependencies:**

```shell
cd /path/to/theme
npm install
```

## Initialize Stencil

**stencil init:**

```shell
stencil init --url <store_url> --token <access_token> --port 3000 --packageManager npm
```

## Start the Live Preview

**Start:**

```shell
stencil start
```
