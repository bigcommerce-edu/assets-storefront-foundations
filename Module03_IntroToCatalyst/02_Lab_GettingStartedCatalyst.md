# Getting Started with Catalyst

## Install the Catalyst Project

**Install:**

```shell
corepack enable pnpm && pnpm create @bigcommerce/catalyst@latest --gh-ref @bigcommerce/catalyst-core@makeswift-canary
```

## Run the Dev Server

**Run:**

```shell
cd /path/to/catalyst/project
pnpm run dev
```
